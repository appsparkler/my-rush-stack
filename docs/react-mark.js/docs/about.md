`react-mark.js` is a text highlighter; a wrapper around the [markjs.io](https://markjs.io/.io) library. It can be used to dynamically mark search terms or custom regular expressions and offers you built-in options like diacritics support, separate word search, custom synonyms, iframes support, custom filters, accuracy definition, custom element, custom class name and more.
