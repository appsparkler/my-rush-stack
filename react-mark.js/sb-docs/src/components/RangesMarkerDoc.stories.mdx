# ⚡️ `<RangesMarker />`

Sometimes you may want to mark text-content based on the
position in the text content instead of a string. For ex.
you would like to mark 5 letters from the 4th position of text and
that's where `RangesMarker` comes to the rescue.

<h2>
  <TargetBlankLink href="https://stackblitz.com/edit/react-mark-js-tqjcef?file=src/examples/RangesExample/index.js">
    Stack Blitz Demo ↗
  </TargetBlankLink>
</h2>

export const args = {
  mark: [
    {
      start: 3,
      length: 5,
    },
  ],
  children: "0123456789",
};

## ‣ Basic Example

In this example, we mark 4 letters starting
from the 3rd letters.

<Canvas>
  <Story
    name="single-range-marker"
